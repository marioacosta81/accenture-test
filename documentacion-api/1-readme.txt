La API se desarrolló sobre el IDE Eclipse STS. También se utilizó el proyecto Lombok, por lo que se requiere configurar el IDE con Lombok para poder compilar el código fuente de la API

El código fuente se entrega también adjunto, comprimido en zip, en el correo de entrega del desarrollo de la prueba  


En la carpeta documentacion-api se encuentran los siguientes archivos:

1. application.properties
Este archivo es el archivo de propiedades de la aplicación. Sirve para observar la configuración de la conexión a la base de datos.
Para el desarrollo de la API se utilizó una base de datos MySql

2. ddl-create-database-MySql.pdf
Este archivo es el diagrama del MER

3. ddl-create-database-MySql.sql
Este archivo es el código del MER

4. Insomnia_request-collections.json
Este archivo es una colección de request de los enpoints desarrollados.
Es un export desde la aplicación Insomnia.
Insomnia se utilizó para probar los request


5. Swagger UI-Accenture-test.pdf
Este archivo contiene información de los enpoints de la API.
Es una impresión de la gráfica de Swagger implementada en la API
